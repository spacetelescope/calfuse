SLA_RANDOM - Random Number   

ACTION:
Generate pseudo-random real number in the range .
CALL:
R = sla_RANDOM (SEED)

GIVEN:

SEED R an arbitrary real number

RETURNED:

SEED R a new arbitrary value
sla_RANDOM R Pseudo-random real number .

NOTE:
The implementation is machine-dependent.
